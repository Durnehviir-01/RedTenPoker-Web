# 红十扑克 - 网页版部署指南

把这个文件夹里的内容上传到任意静态网站托管服务，朋友们就能通过链接直接点开玩了。

## 方案一：GitHub Pages（免费，推荐）

### 步骤 1：注册/登录 GitHub
打开 https://github.com ，没有账号就注册一个。

### 步骤 2：创建仓库
1. 点击右上角 `+` → `New repository`
2. 仓库名称填：`red-ten-poker`（或任何你喜欢的名字）
3. 选择 `Public`（公开）
4. 点击 `Create repository`

### 步骤 3：上传文件
在仓库页面里：
1. 点击 `uploading an existing file` 链接
2. 把本文件夹里的**所有文件**拖进去（index.html、index.js、index.wasm 等）
3. 点击 `Commit changes`

> 注意：index.wasm 文件约 42MB，上传可能需要一点时间。

### 步骤 4：开启 GitHub Pages
1. 在仓库页面点击 `Settings`
2. 左侧菜单点击 `Pages`
3. 在 "Source" 下选择 `Deploy from a branch`
4. Branch 选择 `main`，文件夹选择 `/(root)`
5. 点击 `Save`

### 步骤 5：访问游戏
等待 1-2 分钟后，你的游戏链接就是：
```
https://你的用户名.github.io/red-ten-poker/
```
把这个链接发给朋友即可！

---

## 方案二：Vercel（免费，国内访问稍快）

1. 打开 https://vercel.com ，用 GitHub 账号登录
2. 点击 `Add New...` → `Project`
3. 导入你的 GitHub 仓库（按照方案一先把代码传 GitHub）
4. 直接点击 `Deploy`
5. 部署完成后会给你一个 `xxx.vercel.app` 的链接

---

## 方案三：腾讯云 COS / 阿里云 OSS（国内最快，少量费用）

如果你希望国内朋友访问速度快：
1. 注册腾讯云或阿里云
2. 购买对象存储（COS/OSS），开通静态网站托管
3. 把本文件夹所有文件上传
4. 绑定自定义域名（可选）
5. 分享链接

费用：按流量计费，几个人玩几乎免费。

## 注意事项

1. **浏览器要求**：需要支持 WebGL 2.0 的现代浏览器（Chrome、Edge、Firefox、Safari 均可）
2. **首次加载**：WASM 引擎文件约 42MB，但实际传输时服务器会自动压缩到约 15-20MB，请耐心等待首次加载（之后浏览器会缓存）
3. **单机版**：网页版仅支持单机对战（你与 3 个 AI），不支持联机
4. **手机/平板**：可以用手机浏览器打开，但屏幕较小体验可能不如电脑

## 常见问题

**Q：上传后打开网页黑屏/报错？**
A：请确认上传了所有文件（尤其是 `index.wasm` 和 `index.pck`），并且使用的是现代浏览器。

**Q：链接打开特别慢？**
A：GitHub Pages 在国内访问较慢。如果主要给国内朋友玩，建议使用 Vercel 或国内云存储。

**Q：想更新游戏内容怎么办？**
A：修改游戏后重新导出 Web，然后覆盖上传即可。
