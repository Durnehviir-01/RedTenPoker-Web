# 红十扑克 - 网页版

在线四人红十扑克游戏，支持人机对战。

## 玩法规则

- **牌数**：52 张（无大小王）
- **特殊牌**：红桃 10 (♥10) 和方块 10 (♦10) 是癞子，可以替代任意牌组成牌型
- **组队**：
  - 若两人各有一张红 10，则这两人为队友（2v2）
  - 若一人同时握有两张红 10，则该玩家独自对抗其他三人（1v3）
- **牌型**：单张、对子、三张、顺子、连对
- **胜负**：同一阵营的两人全部出完手牌即获胜

## 在线游玩

👉 [点击开始游戏](https://你的用户名.github.io/RedTenPoker-Web/)

## 操作说明

- **点击手牌**：选中要出的牌
- **出牌**：点击"出牌"按钮
- **不出**：无法或不想跟牌时点击
- **提示**：AI 帮你选择可压过当前牌型的手牌
